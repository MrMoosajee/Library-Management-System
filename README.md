# Library-Management-System
Qt Framework Hardcoded GUI Library Management System. Handle all the management of a Library System. Add, Search, Return and Borrow books/magazines. Handle all file loading and saving and data persistence.

University Qt Framework Course Assignment

Please feel free to add branches of how this can be changed up. Not a big fan of frontend but would love to see different ways of making the visuals on this stand out a bit more, learn a thing or two when it comes to frontend.
All logic done by myself, a little bit of assistance in knowledge gaps thanks to generative-AI and docs.
Qt 5.3 used and c++11
Would love to see this be able to move over to android
